g_or_c

Description

Write a code that prints, for each character of a sequence, a message telling if the character contains G, C or another type of base.

Hint

Examples

Ex1:
    With the sequence ACGT as input the output should be:
    base is not G or C
    base is C
    base is G
    base is not G or C
    
Ex2: 
    With the sequence AACCGG as input the output should be:
    base is not G or C
    base is not G or C
    base is C
    base is C
    base is G
    base is G
