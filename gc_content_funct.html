gc_content_funct

Description

Compute the GC content of various DNA sequence provided by the user. The program should work exactly as gc_content_input, but implementing a function gc_content() that computes one single sequence at a time. The function should pass the tests in the Tests section.

Hint

The part of the code related to the input() call should be placesd outside gc_content(). Use: 

if __name__ == "__name__":

Tests

    >>> gc_content("GCGAGCTAGCTAGCGATCAGCATCATCACATGCATCGTGTA")
    51.21951219512195
    >>> gc_content("CGCGCGCGCCCGCGCGCGCGCGCCCCGGGCGCGCGCGCGCGCG")
    100.0
    >>> gc_content("")
