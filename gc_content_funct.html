gc_content_funct

Description

Compute the GC content of various DNA sequence provided by the user. The program should take data from the user using the input() funtion. First, the user will type an integer corresponding to the number of sequence to compute the GC content. Then, it will compute and display the GC content of each of them. The code should do the same taks as "gc_content_input" but implement a function gc_content(seq) to compute the GC content each individual sequence. The function should pass the tests in the Tests section.

Hint

For easier testing of the gc_content() function, put the code outside gc_content() following
if __name__ == "__name__":

Tests

    >>> gc_content("GCGAGCTAGCTAGCGATCAGCATCATCACATGCATCGTGTA")
    51.21951219512195
    >>> gc_content("CGCGCGCGCCCGCGCGCGCGCGCCCCGGGCGCGCGCGCGCGCG")
    100.0
    >>> gc_content("")
