tetrapeptides

Description

Write a code that, given a string with residues, generates a list (and prints it) with all possible tetrapeptides that can be constructed combining the residues in the string. Repetition is possible.

Hint

Examples

Ex1:
    With the residues A L as input, the output should be:
    AAAA
    AAAL
    AALA
    AALL
    ALAA
    ALAL
    ALLA
    ALLL
    LAAA
    LAAL
    LALA
    LALL
    LLAA
    LLAL
    LLLA
    LLLL
    
Ex2: 
    With the residue A as input, the output should be:
    AAAA
