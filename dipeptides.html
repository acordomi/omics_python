dipeptides

Description

Write a code that, given a string with residues, generates a list (and prints it) with all possible dipeptides that can be constructed combining the residues in the string. Repetition is possible.

Hint

Examples

Ex1:
    With the residues A Y S as input, the output should be:
    AA
    AY
    AS
    YA
    YY
    YS
    SA
    SY
    SS
    
Ex2: 
    With the residue L as input, the output should be:
    LL
