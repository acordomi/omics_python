join_cds

Description

The file brca1_cds.dna contains the coding sequence of BRCA1 (each line corresponds to a coding exon). Can you reconstruct the whole CDS in a single string? We would like to get a file as output with the joined CDS (in one line) in RNA alphabet and uppercase letters.

Hint

Main tasks: read a file, iterate the lines, update the accumulator, write a file.

The str methods .replace(), .upper() and .strip() may be of help

Examples
