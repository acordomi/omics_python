rna_to_prot

Description

Write a script that reads RNA sequences and translates them to protein alphabet (you can finde the standard genetic code here). You will need two functions: 1) read_gencode(xxx): should read the file and return a dictionary with the genetic code. 2) rna2prot(xxx): should take list of triplets and return a string with the protein sequence.

Tests

    >>> len(read_gencode("gencode.txt"))
    64
    >>> rna2prot(["AUU", "CAU", "AUC"])
    'IHI'
    >>> rna2prot(["ATT", "CAT", "ATC"])
    'IHI'
    >>> rna2prot(["CGA", "GUG", "UAA"])
    'RV/'
