score_seqs_gap

Description

Write a function score_seqs_gap(seq1, seq2, match, mismatch, gap) that computes the alignment score between two sequences (seq1 and seq1) using a match, a mismatch and a gap penalty score. Both sequeces should have the same lenght; otherwise the score should be 0.

Tests

    >>> score_seqs("THEFASTCAT", "THEFATCAT ", 1, -1, 0)
    1
    >>> score_seqs("THEFASTCAT", "THEFATCA T", 1, -1, 0)
    3
    >>> score_seqs_gap("THEFA TCAT", "THEFASTCAT", 1, -1, 0)
    9
    >>> score_seqs_gap("THEFASTCAT", "THE", 1, -1, 0)
    0
