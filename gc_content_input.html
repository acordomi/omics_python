gc_content_input

Description

Compute the GC content of various DNA sequence provided by the user. The program should take data from the user using the input() funtion. First, the user will type an integer corresponding to the number of sequence to compute the GC content. Then, it will compute and display the GC content of each of them.

Hint

Examples

Ex1:
The input is:
3
GCGAGCTAGCTAGCGATCAGCATCATCACATGCATCGTGTA
CGCGCGCGCCCGCGCGCGCGCGCCCCGGGCGCGCGCGCGCGCG
AAATATATA
  
The output should be:
GC content (%): 51.2
GC content (%): 100.0
GC content (%): 0.0

Ex2: 
The input is:
2

AGCTATCATCTAT

The output should be:
emtpy sequence
GC content (%): 30.8
