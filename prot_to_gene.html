prot_to_gene

Description

Write a function prot2gene(protein_name) that, given a protein entry name, returns its (standard) gene name.

Hint

You may use Biopython's SwissProt tools or create your own

Tests

    >>> prot2gene("PACR_BOVIN")
    'ADCYAP1R1'
    >>> prot2gene("")
    ''
    >>> prot2gene("AAAAA_HUMAN")
    ''
