gc_content_cds

Description

The file brca1_cds.dna contains the coding sequence of BRCA1 (each line corresponds to a coding exon). Compute the GC content of each exon. We would like to get: 1) a list with the GC content of each of the exons; 2) the exons with the minimum and maximum GC content.

Hint

Main tasks: read a file, iterate the lines, compute GC content, update the accumulator.

The functions min() and max(), that operate on lists, may be of help.

Examples

Ex1:

  With the provided file, the output should be:
  GC content of each line: [37.5, 42.593, 37.179, 38.202, 37.143, 46.226, 36.957, 41.558, 39.025, 49.438, 47.093, 40.157, 46.597, 47.91, 35.227, 39.744, 39.024, 46.429, 52.727, 48.649, 55.738, 57.6]
  Minimum: 35.227    Maximum: 57.6
