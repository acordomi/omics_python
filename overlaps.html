overlaps

Description

Write a fuction count_ov(sequence, motif) that works similar to str.count() method but is aware of overlaps between motifs (see Tests).

Hint

Tests

    >>> 'GATATAT'.count('ATAT')
    1
    >>> count_ov('GATATAT', 'ATAT')
    2
    >>> count_ov('GATATATATATGCATATATACTT', 'ATAT')
    6
    >>> count_ov('GAT', 'ATAT')
    0
    >>> count_ov('GAT', '')
    0
