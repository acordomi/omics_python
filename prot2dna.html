prot2dna

Description

Write a function prot2dna(seq) that uses the back_table dictionary in CodonTable to generate one possible DNA sequence ncoding the input sequence.

Hint

import CodonTable from Bio.Data
The dictionary is in CodonTable.unambiguous_dna_by_name['Standard'].back_table

Tests

    >>> len(read_gencode("gencode.txt"))
    64
    >>> rna2prot(["AUU", "CAU", "AUC"])
    'IHI'
    >>> rna2prot(["ATT", "CAT", "ATC"])
    'IHI'
    >>> rna2prot(["CGA", "GUG", "UAA"])
    'RV/'
