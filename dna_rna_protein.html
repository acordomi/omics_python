dna_rna_protein

Description

Write a code that detects if a sequence is a DNA, an RNA or protein. Assume that sequences with only A, C, T and G are DNAs. Do not consider ambiguous amino acids or bases.

Hint

If a sequence contains 'U' it must be RNA. If it contains characters other than ACTGU it is a protein. It is possible to break a for loop prematurely using break

Examples

Ex1:
ACTGTGTAA is a DNA
    
Ex2: 
ACUGCUAG is an RNA

Ex3: 
AHPLCTGA is a protein
