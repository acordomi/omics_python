dna2allprots

Description

Write a function that returns the list all possible proteins that could be encoded from a DNA segment. Consider the 3 reading frames both on the forward and the reverse complementary sequence.

Hint

You can open the fasta file using Biopython 

from Bio import SeqIO
SeqIO.parse(filename, "fasta")

Tests

>>> dna2allprots('AGTCGATGCACACGATCGATGCATCGATATCGTATATCCGATACCCCGTAC')
    ['SRCTRSMHRYRISDTPY', 'VDAHDRCIDIVYPIPR', 'SMHTIDASISYIRYPV', 'VRGIGYTISMHRSCAST', 'YGVSDIRYRCIDRVHR', 'TGYRIYDIDASIVCID']
