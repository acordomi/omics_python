gc_content

Description

Compute the GC content of a DNA sequence defined in a variable.

Hint

Use the function round() to present only two decimals

Examples

    Ex1:
    With seq = "GCGAGCTAGCTAGCGATCAGCATCATCACATGCATCGTGTA" the program should print:
    GC content (%): 51.2

    Ex2:
    With seq = "CGCGCGCGCCCGCGCGCGCGCGCCCCGGGCGCGCGCGCGCGCG" the program should print
    GC content (%): 100.0

    Ex3:
    With seq = "AAATATATA" the program should print
    GC content (%): 0.0
